# Multi-Task-Learning-with-Limited-Labels

This project aims to implement Multi-Task Learning models with limited labels used for Master Thesis.
## Requirements

To run this project, you will need the following dependencies:

*   python==3.8.19
*   numpy>=1.24.4
*   torch==2.1.2
*   torchvision==0.16.2
*   scipy==1.10.1
*   timm==1.0.7
*   wandb==0.16.6
